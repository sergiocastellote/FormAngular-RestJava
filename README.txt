Aplicación web construida con AngularJS para el front y Java para el backend


Para arrancar front:

-Entrar en el directorio donde estén los archivos package.json y bower.json.
-Ejecutar npm install
-Ejectuar bower install
-Arrancar proyecto con gulp-serve


Para arrancar back:

-Importar proyecto en eclipse
-Crear servidor Tomcar 8.0 y añadir proyecto
-Arrancar servidor


BBDD:

-Instalar BBDD MySql
-Crear tabla 'users' con campos 'name' y 'surname'
